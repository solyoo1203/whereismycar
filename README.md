# 내 차 여기 - 주차 위치 기록 웹앱

손그림 SVG 디자인을 그대로 사용하는 모바일 웹앱/PWA입니다.

## 기능

- 기둥 사진 촬영
- 위치 메모 저장
- 지상/지하/층수 저장
- 주차 시작 후 실시간 타이머
- 출차 완료 시 현재 기록 삭제
- 장기주차용 저장 목록
- 서버 없이 브라우저 `localStorage`에 저장
- 사진은 휴대폰 사진첩에 저장하지 않고, 브라우저 데이터 안에만 저장

## GitHub Pages에 올리는 법

1. 이 폴더 안의 파일을 GitHub 저장소에 업로드합니다.
2. GitHub 저장소에서 `Settings` → `Pages`로 이동합니다.
3. `Branch`를 `main` / `/root`로 선택하고 저장합니다.
4. 잠시 후 표시되는 주소로 접속합니다.

## 디자인 파일 교체

디자인 SVG는 `public/assets` 안에 있습니다.
파일명만 그대로 유지하면 SVG를 다시 바꿔도 앱에 자동 반영됩니다.

## 주의

브라우저 데이터 삭제, 시크릿 모드 사용, 앱 캐시 삭제 시 저장된 주차 기록이 사라질 수 있습니다.
